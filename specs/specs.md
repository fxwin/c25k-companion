Spec: Web App Companion for the Couch to 5k Program https://c25k.com/, to be used on a mobile device.
Environment: VPS with nginx installed and a domain already registered at fxwin.net
Requirements: Minimal Web application, hosted at fxwin.net/c25k, with the following features:
The program consists of 3 Workouts per week, each consisting of different sections like warmups, jogging, and walking, possibly repeated.
As the user, i want to open the webpage, see a "Recommended Workout" and a Start/Stop/Restart Button. Once the Workout is finished, I should be able to mark it as completed, in which case the next workout becomes the "Recommended Workout".
The performed Workouts should be logged with times, dates, and a "how did you feel" score from 1(bad)-5(great).
The first time a user opens this page, they can select three days of the week as "default" workout days, which will be used to display a "next workout: in 2 days/tomorrow/today".
Store information in the browsers localstorage, no need for a separate database or logins.
Before starting a workout, i should see a timeline of components in this workout. 
After starting a workout, i should see my progress through this timeline.
During the workout, keep track of the time with an on-screen timer.
